## Project overview

一个超市结算系统，支持苹果、草莓、芒果的定价与促销策略，包含以下场景：
- **场景 1：** 基础定价（苹果 ¥8/斤，草莓 ¥13/斤）。
- **场景 2：** 增加芒果（¥20/斤）。
- **场景 3：** 草莓限时八折。
- **场景 4：** 满 ¥100 减 ¥10。

通过面向对象设计实现商品、购物车、折扣策略的解耦，并提供可执行的结果验证。

---

## Features

- **面向对象设计：** 商品、购物车、策略类职责清晰，易扩展。
- **灵活的促销策略：** 支持百分比折扣、满减等组合。
- **结果验证脚本：** 一键运行对函数计算结果进行校验。
- **可扩展：** 轻松新增水果或促销规则。

## Quick start

- **环境准备：**
  ```bash
  conda create -n market python=3.8
  conda activate market
  ```

- **验证计算结果：**
  ```bash
  python evaluation.py
  ```
---

## File structure

- **12.29market.py：** 核心结算与策略实现（商品、购物车、折扣策略）。
- **evaluation.py：** 对各场景的函数结果进行断言验证，保证正确性。
- **main.py：** 演示入口，打印各场景的结算结果。
- **q1.py / q2.py / q3.py / q4.py：** 各题目独立实现或演示（可选）。
- **test.py：** 其他临时测试（可选）。
