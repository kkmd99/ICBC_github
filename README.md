## Project overview

一個超市結算系統，支援蘋果、草莓、芒果的定價與促銷策略，包含以下場景：
- **場景 1：** 基礎定價（蘋果 ¥8/斤，草莓 ¥13/斤）。
- **場景 2：** 增加芒果（¥20/斤）。
- **場景 3：** 草莓限時八折。
- **場景 4：** 滿 ¥100 減 ¥10。

透過物件導向設計實現商品、購物車、折扣策略的解耦，並提供可執行的結果驗證。

---

## Features

- **物件導向設計：** 商品、購物車、策略類職責清晰，易擴充套件。
- **靈活的促銷策略：** 支援百分比折扣、滿減等組合。
- **結果驗證指令碼：** 一鍵執行對函式計算結果進行校驗。
- **可擴充套件：** 輕鬆新增水果或促銷規則。

---

## Quick start

- **環境準備：**
  ```bash
  conda create -n market python=3.8
  conda activate market
  ```

- **驗證計算結果：**
  ```bash
  python evaluation.py
  ```
- **執行上面程式碼後輸出結果為:**
  ```python
  測試1: 期望218.4，實際 218.4
  測試2: 期望109.2，實際 109.2
  所有測試透過 ✔
  ```
